Styling Symbols

Styling is limited to properties that inherit, or the SVG element itself.


          svg {
            fill: DarkOrchid;
            transition: fill 250ms;

            &:hover {
              fill: MediumVioletRed;
            }
          }
        

          height: 50px;
          width: 50px;
          transition: transform 250ms;

          &:hover {
            transform: rotate(180deg);
          }
